# 固定资产领域词典

本文记录固定资产质检 Agent 使用的字段、业务口径和取值约束。若业务口径发生变化，先更新本文，再调整规则实现。

## 核心字段

| 字段 | 建议字段名 | 含义 | MVP 校验要求 |
| --- | --- | --- | --- |
| 资产编码 | `asset_id` | 固定资产唯一标识 | 必填、唯一、不可重复 |
| 资产名称 | `asset_name` | 资产展示名称 | 必填 |
| 资产类别 | `asset_category` | 资产分类，如房屋建筑物、机器设备、电子设备 | 必填，后续维护枚举 |
| 原值 | `original_value` | 资产入账原始价值 | 必填，数值大于等于 0 |
| 累计折旧 | `accumulated_depreciation` | 截至质检时累计折旧金额 | 必填，数值大于等于 0 |
| 净值 | `net_value` | 原值扣减累计折旧后的账面价值 | 必填，通常等于原值减累计折旧 |
| 减值准备 | `impairment_provision` | 固定资产已计提减值准备 | 建议字段；金额关系校验需要 |
| 使用寿命（月） | `useful_life_months` | 固定资产折旧使用寿命，按月表示 | 折旧测试必填，需为正数 |
| 残值率 | `salvage_rate` | 预计净残值占原值比例 | 折旧测试必填，通常在 0 到 1 之间 |
| 本期计提折旧 | `current_depreciation` | 本期账面记录的折旧计提金额 | 折旧测试必填 |
| 新增方式 | `addition_method` | 资产本期新增来源，如购置、在建工程转入、企业合并 | 新增测试必填 |
| 处置日期 | `disposal_date` | 资产处置、报废或减少日期 | 处置测试必填 |
| 处置/减少方式 | `disposal_method` | 资产减少类型，如出售、报废、其他减少 | 处置测试必填 |
| 使用部门 | `department` | 当前使用或管理资产的部门 | MVP 可选，后续接组织主数据 |
| 存放地点 | `location` | 资产实际存放位置 | MVP 可选，后续接地点主数据 |
| 启用日期 | `start_date` | 资产开始使用或入账日期 | 可选；如果存在，不应晚于质检日期 |
| 使用状态 | `status` | 在用、闲置、报废、处置中等状态 | MVP 可选，后续维护枚举 |

## 底稿与审计术语

| 术语 | 含义 | Agent 使用方式 |
| --- | --- | --- |
| `K.00 Lead Sheet` | 固定资产两期变动引导表 | 检查基础信息、预期分析、异常波动和阈值变化 |
| `K.01 Agree SL to GL` | 明细账/后推明细表与总账核对程序 | 检查后推明细表、总账、资产清单和试算表之间是否核对 |
| `FA List` | 固定资产明细清单 | 作为资产级字段、金额关系、折旧测试的基础数据 |
| `K.02.1 Additions` | 新增资产测试 | 检查新增清单、后推表核对、总体同质性和新增证据 |
| `K.02.2 Disposals` | 处置/报废测试 | 检查处置清单、后推表核对、处置完整性和总体同质性 |
| `K.03.1 SAP` | 折旧实质性分析程序 | 判断中/高精确度 SAP 适用性和是否需要补充 TOD |
| `K.03.2 TOD` | 折旧详细测试 | 重新计算折旧并与账面折旧比较 |
| `K.03.3 折旧政策复核` | 折旧方法、使用寿命、残值率复核 | 多数需要人工复核 |
| `TE` | 可容忍误差 | 程序范围、差异调查和阈值判断的重要参数 |
| `SAD` | 名义金额或差异汇总门槛 | 差异超过该金额通常需要调查 |
| `TT` | 测试阈值 | 决定测试范围或样本总体的重要参数 |
| `CRA` | Combined Risk Assessment | 影响 SAP 精确度、证据水平和是否需要 TOD |
| `PSP` | 规定的实质性程序 | 检查是否执行或是否有充分拒绝执行理由 |
| `OSP` | 其他实质性程序 | 盘点、权属、减值等可作为后续扩展 |

## 质检结果

| 结论 | 含义 | 处理方式 |
| --- | --- | --- |
| `PASS` | 校验通过 | 无需处理 |
| `WARN` | 存在轻微风险 | 建议业务确认 |
| `FAIL` | 明确不符合规则 | 需要修复数据或补充材料 |
| `NEED_REVIEW` | 自动规则无法判断 | 进入人工复核 |

## MVP 首批规则

| 规则 | 说明 | 默认结论 |
| --- | --- | --- |
| 必填字段校验 | 核心字段为空时返回问题 | `FAIL` |
| 资产编码唯一性 | 同一台账中资产编码重复 | `FAIL` |
| 金额关系校验 | `original_value - accumulated_depreciation - impairment_provision` 应约等于 `net_value` | `FAIL` |
| 金额非负校验 | 原值、累计折旧、净值不得为负 | `FAIL` |
| 日期合理性校验 | 启用日期不应晚于质检日期 | `WARN` |
| 使用寿命校验 | 使用寿命（月）应为正数 | `FAIL` |
| 残值率校验 | 残值率通常应在 0 到 1 之间 | `FAIL` |
| 折旧重新计算 | 按政策或基础公式重算本期折旧，与账面折旧比较 | `WARN` |

## 待确认业务口径

- 金额关系是否允许小数误差，建议 MVP 先按 `0.01` 作为容差。
- 资产类别枚举由财务系统、ERP 还是人工维护。
- 报废、处置中资产是否继续参与全部规则。
- 后续影像材料、发票、合同与台账如何建立关联。
