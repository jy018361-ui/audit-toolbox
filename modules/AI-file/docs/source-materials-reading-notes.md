- 固定资产实操指引的使用说明
- 1、本实操指引仅适用于固定资产科目评估为重大账户时使用，若固定资产科目评估结果为重大账户以外的账户性质（如：有限风险账户），请参考固定资产有限风险账户实操指引或结合风险评估的结果执行审计程序；  2、本实操指引内容仅涉及除<K-05-GL: 固定资产的减值>外的PSP，不含固定资产的OSP。  3、本实操指引的内容主要包括基础操作指引、进阶实操提示、易错点。“基础操作指引”旨在让使用者快速上手执行审计程序;“进阶实操提示”补充了不同业务场景下的相关考虑及理论依据；“易错点”归纳了审计程序执行过程中的常见错误。  4、本实操指引源于审计实务经验的归纳和总结，不能代替 GAM 及审计准则。 审计项目组可结合审计项目的业务特征、风险评估结果、所处监管坏境等因素参考使用。  5、审计过程中涉及到的职业判断需要由审计项目组完成，实操指引的内容不作为任何职业判断依据，仅供参考。  6、本实操指引仅涵盖SWP（excel）内容，审计项目组需确保在SWP中执行的审计程序与Canvas中对应的工作程序表（canvas form）内容一致。 | 若在实操指引使用中有任何问题，欢迎大家踊跃扫描二维码或者点击链接提出意见和建议。
