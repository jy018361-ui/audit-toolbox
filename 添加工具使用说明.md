# 添加工具使用说明

## 快速开始

### 方法一：使用添加工具界面（推荐）

1. **双击运行 `添加工具.bat`**
   - 会打开一个图形界面

2. **填写工具信息**
   - **工具名称**：输入工具的中文名称（如"发票识别工具"）
   - **工具描述**：简单说明工具用途（可选）
   - **选择入口脚本**：点击"浏览..."选择你的主文件（.py 文件）
   - **额外文件夹**：如果工具有其他依赖文件夹（如 gui/ 目录），可以选择

3. **点击"添加工具"**
   - 工具会自动复制到 `modules/` 目录（默认不入主仓库 Git）
   - `tools.json` 会自动更新

4. **重启工具箱**
   - 运行 `启动审计工具箱.bat` 或 `python suite_main.py`
   - 新工具会显示在列表中

### 方法二：手动添加

如果你熟悉命令行，可以手动操作：

#### 步骤 1：创建工具目录

```
modules/
└── your_tool/        # 与 tools.json 中 vendor_dir 一致
    ├── main.py       # 入口文件（必需）
    ├── gui/          # GUI 目录（可选）
    └── other.py      # 其他文件（可选）
```

#### 步骤 2：编写入口文件

入口文件必须包含一个 `main()` 函数，支持 `parent` 参数：

```python
# modules/your_tool/main.py
"""
我的审计工具
"""
import tkinter as tk
from tkinter import ttk, filedialog, messagebox


def main(parent=None):
    """
    工具入口函数。
    
    Args:
        parent: 父窗口（tk.Tk 或 None）
    """
    # 创建窗口
    if parent is not None:
        root = tk.Toplevel(parent)
        root.transient(parent)
    else:
        root = tk.Tk()
    
    root.title("我的工具")
    root.geometry("900x600")
    
    # === 在这里添加你的 GUI 代码 ===
    ttk.Label(root, text="工具加载成功！").pack(pady=20)
    # === 结束 ===
    
    root.mainloop()


if __name__ == "__main__":
    main()
```

#### 步骤 3：更新 tools.json

在 `tools.json` 的 `tools` 数组中添加配置：

```json
{
  "id": "your_tool",
  "name": "我的工具",
  "description": "工具功能描述",
  "vendor_dir": "your_tool",
  "entry": "main.py",
  "callable": "main"
}
```

#### 步骤 4：提交代码

```bash
# 独立仓库模式：只提交 tools.json；源码留在你的仓库或本地 modules/
git add tools.json
git commit -m "feat: 注册 XXX 工具"
git push
```

## 常见问题

### Q: 为什么我的工具没有显示在列表中？

A: 检查以下几点：
1. `tools.json` 中是否正确添加了工具配置
2. `vendor_dir` 是否与 `modules/`（或 `tools/`）下的目录名一致
3. 是否已执行 `python build_suite.py --sync-only`（若依赖 vendor 运行）
4. 入口文件是否存在
5. 重启工具箱

### Q: 如何添加额外的 Python 依赖库？

A: 需要修改以下文件：

1. **requirements.txt** - 添加依赖包：
```
your_new_library>=1.0
```

2. **launcher/bundle_anchor.py** - 添加 import：
```python
def touch_bundle_deps() -> None:
    import dateutil
    import numpy
    import openpyxl
    import pandas
    import polars
    import python_calamine
    import xlsxwriter
    import xlrd
    import your_new_library  # 添加这一行
```

3. **suite.spec** - 添加到 hiddenimports：
```python
hiddenimports = [
    # ... 现有列表 ...
    "your_new_library",
]
```

### Q: 工具独立运行正常，但集成到工具箱后报错？

A: 可能是路径问题。确保入口文件支持 `parent` 参数：

```python
def main(parent=None):
    if parent is not None:
        root = tk.Toplevel(parent)
        root.transient(parent)
    else:
        root = tk.Tk()
    # ...
```

### Q: 如何调试工具加载问题？

A: 在命令行运行查看错误信息：
```bash
python suite_main.py
```

```bash

git pull

# 同步工具到 vendor
python build_suite.py --sync-only

python build_suite.py

# 输出位置
dist/审计工具箱.exe
```

## 注意事项

1. **编码**：所有 Python 文件使用 UTF-8 编码
2. **GUI**：使用 tkinter 和 ttk 组件
3. **入口函数**：必须支持 `parent` 参数
4. **依赖**：如有新依赖，需更新 requirements.txt、bundle_anchor.py 和 suite.spec
5. **测试**：提交前测试工具可独立运行
